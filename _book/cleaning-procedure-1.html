A Minimal Book Example

Chapter 5 Cleaning Procedure

  • Step 1: Thoroughly review all survey documentation Examine the raw data files, questionnaires (all versions), interviewer manuals, sampling guidelines, codebooks (if any exist), and any field reports or data entry notes. The goal is to achieve a complete understanding of the survey structure, question wording changes over time, skip patterns, coding conventions, and any known data issues.

  • Step 2: Create comprehensive codebooks and metadata For each survey year/round, produce a detailed data codebook that includes:

    • Variable name
    • Full question text
    • Response categories and numeric codes
    • Universe (who was asked the question)
    • Missing value codes and their meanings
    • Any recording or cleaning notes

    Additionally, create a master metadata file (harmonization crosswalk) that maps variables across years, documenting:

    • Variables that are identical
    • Variables that changed wording or response options
    • Variables that were added or dropped
    • Recommended harmonized variable names and labels

    This metadata file is the single most important document for longitudinal consistency.

  • Step 3: Clean and harmonize the data Produce two levels of cleaned datasets:

    • Year-specific cleaned versions: fully cleaned individual-year datasets with consistent naming conventions within each year, detailed value and variable labels, and thorough documentation of all cleaning decisions.
    • Master harmonized dataset: a single appended (or panel) dataset containing all years, with harmonized variable names, comparable categories, and flags indicating when values have been recorded or imputed for comparability.

    Note: All cleaning and harmonization must be done exclusively via reproducible scripts (Stata do-files, R scripts, Python notebooks, etc.), never by manual editing of the data.

  • Step 4: Finalize documentation and archiving Prepare the complete data package for long-term preservation and sharing:

    1. Original raw data (preserved unchanged)
    2. Survey questionnaires and interviewer guidelines (all versions)
    3. Year-specific codebooks
    4. Master metadata/harmonization crosswalk
    5. Full set of reproducible scripts/do-files that generate both the year-specific and master harmonized datasets from the raw data
    6. Year-specific cleaned datasets
    7. Final master harmonized dataset
    8. Detailed data correction log documenting every issue encountered (outliers, inconsistencies, coding errors, variable changes) and exactly how each was resolved
